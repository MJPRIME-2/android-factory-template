# M2 Solutions - Android Factory Template

Standard template for all apps built by the M2 Solutions App Factory.

## Tech Stack
- Kotlin + Jetpack Compose
- Material Design 3 (Dynamic Color)
- Hilt (Dependency Injection)
- Room (Local Database)
- Min SDK 26 (Android 8.0)

## Usage
This repo is used as a GitHub template. New apps are created from it automatically by the factory pipeline.

## CI/CD
- **build.yml** — Builds debug APK and runs tests on push/PR
- **release.yml** — Builds signed release APK/AAB on version tags
